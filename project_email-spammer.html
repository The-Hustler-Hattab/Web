Email Spammer

Send bulk emails or sms to any person

This is a simple API project designed to facilitate email and SMS spamming. It is built using Flask. The application allows users to send bulk emails or text messages to multiple recipients directly from the web interface.

Key Components:

  • Python Flask Framework
  • Mysql
  • Internal email and SMS sending mechanism

Project Overview:

The application is hosted on Google Cloud Run, a serverless platform, which provides scalability and ease of management.

Deployment of the application is automated, enabling efficient management of changes and updates.
